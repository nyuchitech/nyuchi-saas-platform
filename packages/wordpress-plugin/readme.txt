== Description ==

Nyuchi Platform Connector is a lightweight plugin that connects your WordPress site to the Nyuchi Platform - a premier SaaS SEO management system.

**How it works:**
1. Install and activate the plugin
2. Get your API key from dashboard.nyuchi.com
3. Enter the API key in plugin settings
4. Manage all SEO operations from the Nyuchi Platform dashboard

This plugin is designed to be lightweight and secure, handling only the communication between your WordPress site and the Nyuchi Platform.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/nyuchi-platform-connector/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings ‚Üí Nyuchi Platform to configure the connection
4. Enter your API key from dashboard.nyuchi.com
5. Start managing your SEO from the Nyuchi Platform dashboard

== Frequently Asked Questions ==

= What is Nyuchi Platform? =
Nyuchi Platform is a comprehensive SaaS SEO management system that allows you to manage SEO for unlimited WordPress sites from a single dashboard.

= Do I need a Nyuchi Platform account? =
Yes, you need to create an account at dashboard.nyuchi.com to use this plugin.

= Is this plugin free? =
The plugin is free to install. Nyuchi Platform offers various pricing plans for the SaaS service.
